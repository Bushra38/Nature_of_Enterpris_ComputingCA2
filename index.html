Powerlifting Dataset Analysis

Exploratory analysis of lifter performance using descriptive statistics, visualisation, and a Linear Regression model to predict Wilks score.

Dataset Overview

This project analyses powerlifting competition results including Age, Bodyweight, Best Squat, Best Bench Press, Best Deadlift, Total Lift, and Wilks score. The goal is to identify performance trends and apply machine learning to predict relative strength (Wilks).

Data Cleaning

  • Removed rows with missing or invalid values in key columns (Age, Bodyweight, lifts, Total, Wilks).
  • Converted numeric columns from text to numbers for accurate calculations.
  • Checked for duplicate records to avoid double counting.
  • Reviewed extreme values to ensure realistic performance data was retained.

After cleaning, the dataset was prepared for descriptive statistics, visual analysis, and regression modelling.
